Deploy no Vercel
🚀 Como publicar no Vercel
1
Instala o Node.js e o Vercel CLI:
npm install -g vercel
2
Entra na pasta do projeto e faz login:
cd quantbot vercel login
3
Faz o deploy com um comando:
vercel --prod
4
O Vercel dá-te um URL como quantbot-ia.vercel.app — abre no telemóvel e o painel está online!
5
O bot Python envia os dados reais via POST /api/signals com a função webhook.
